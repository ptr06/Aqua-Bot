Command
Description
setlevel
Sets the channel for level up messages to be sent to
levelreset
Resets all of the levels for everyone in the server
dellevel
Deletes the channel from our database, and stops sending new level up messages
levelchannel
Shows the current channel for leveling messages
setmute
Sets the role that will be given to users whenever you use the `/mute` command
delmute
Deletes the muted role from our database
muterole
Sends the current role that is assigned as the muted role for your server
setjoin
Sets the channel for messages to be sent whenever a new user joins your server
deljoin
Deletes the channel from our database, and stops sending new user messages
joinchannel
Sends the current channel that is assigned as the new user messages channel

Music
Description
play
Searches YouTube, and then plays the top song - You can also send a link
skip
Skips the song that is currently playing
queue
Sends all of the songs that are in the queue
remove
Removes the specified song from the queue
stop
Stops music, clears queue, and leaves VC
clear
Removes ALL songs in the queue
repeat
Repeats the song that is playing. Run the command again to stop repeating
shuffle
Will play a random song in the queue. Run the command again to stop shuffling
np
Sends the song that is currently playing
pause
Pauses the currently playing song
resume
Resumes the paused song

Command
Description
warn
Warn a member for doing something against the rules
delwarn
Delete a warning from a member so that it is no longer on their record
warnings
See all of the warnings for a member. Also includes when they were warned, and who warned them
mute
Mute a member so they can't send anymore messages
tempmute
Mutes the member temporarily, they will be unmuted once the specified time has passed
unmute
Unmute a member so they are able to send messages again
purge
Delete messages from your server. Max amount that can be deleted at one time is `100` messages
kick
Kick a member from your server. They will be able to join back with a new invite
ban
Ban a member from your server. They will not be able to join back until they are unbanned
softban
This command will ban and then immediately unban the member in order to get rid of their message history
